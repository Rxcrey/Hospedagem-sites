Hospedagem de Sites

  • Criar um repositorio no remoto (github)
  • Criar repositorio local: git init
  • Adicionar arquivos em zona de preparação
  • Criar ponto na historia: git commit -m "firt commit"
  • Renomear a branch para main: git branch -M main
  • Unificar repositorios: git remote add origin [endereço-repositorios github]
  • Enviar o codigo: git push -u origin main
